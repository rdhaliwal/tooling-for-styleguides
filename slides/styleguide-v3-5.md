## Distribution problem solved!

Create a bower.json file like this:
```json
{
    "name": "some-app",
    "version": "1.0.0",
    "dependencies": {
        "styleguide": "2.2.0",
        "normalize-css": "x.x.x",
        "angular": "x.x.x",
        "underscore": "x.x.x"
    }
}
```

Then just run
```bash
bower install
```
and done!
